About the Fake Image Detector

This web application uses a convolutional neural network (CNN) to classify whether an uploaded image is AI-generated or real.

Model Accuracy Metrics

The model's performance is evaluated using several metrics derived from test data:

  • Precision: Proportion of predicted AI-generated images that are truly AI-generated.
  • Recall: Proportion of all AI-generated images that were correctly identified.
  • F1-score: A balance between precision and recall, useful for uneven class distributions.
  • Confusion Matrix: Summarizes correct vs. incorrect predictions across both classes.
